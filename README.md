# Focus - YouTube Content Filter

A Chrome extension that blocks distracting YouTube content to help you stay focused.

## What it does

- **Blocks**: Gaming, Entertainment, Comedy, Vlogs, Shorts
- **Allows**: Education, Music, News, Science & Technology

## Installation

1. Clone this repo
2. Go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder
5. Click the extension icon and set up your YouTube API key

## Get Your API Key

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a project and enable **YouTube Data API v3**
3. Create an API key
4. Enter it in the extension setup page

## Features

- Smart channel learning (caches decisions)
- Edit learned channels (allow/block)
- Blocks all YouTube Shorts
